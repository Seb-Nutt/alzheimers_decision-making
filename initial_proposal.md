# CS 30 Major Project Proposal

## Description
    I am hoping to create an artificial intelligence model over the span of the next two months. I will combine this with the research on morality in my Research methodology class to attempt to create an explicitly moral AI model. The use of javascript, HTML, and CSS will be used in creating a UI for interacting with the AI chatbot. I will examine and moderate the machine learning process to examine its effects on the final result. I will be developing and coding the AI model myself but may use outside resources to provide guidance.

## Needs to have

### Frontend:
 - A working user interface, one of the following:
    - A 2d UI similar to that of ChatGPT
    - A 3d space that allows for movement and teh AI in a fixed location that you can interact with
 - Customizible colors
 - Text to speech

### Backend:
 - An machine learning model that is able to carry a convesation with the user
 - The ability to remember and learn from the user
 - Use either implict or explicit moral values to act as a moral agent

## Nice to have
 - Image/Video support
 - Text to speech and speech to text systems
 - The ability to import training data to see the results on the model
 - Saving past conversations
 - An error catching system that either resolves or logs errors when they are present
 - Preset moral dilemmas that allow for testing the effects of different collections of training data
